# Advanced Programming Coursework

This repository contains my coursework submissions for the **Advanced Programming** course. The assignments cover a wide range of programming paradigms, languages, and software development practices, including system-level programming in C, object-oriented programming in Java and Python, and modern frontend web development using React.

## 📂 Repository Structure

The repository is organized by assignment number. Below is a thematic breakdown of the topics and languages covered in each assignment:

### 🧩 C Programming
Focuses on core computer science concepts, memory management, and algorithmic complexity.
* **Assignment 1 & 2:** Time and Space Complexity analysis (`time_complexity.c`, `space_complexity.c`).
* **Assignment 13, 15, & 16:** Advanced C programming concepts and problem-solving.

### ☕ Java Programming
Explores Object-Oriented Programming (OOP) principles, data structures, and Java Streams.
* **Assignment 3:** `ArrayListBookSearch` - Working with Java Collections.
* **Assignment 6:** `StudentPerformanceAnalyzer` - Time profiling, Java Streams, and OOP.
* **Assignment 9:** `BankingSystem` - Complex object-oriented system modeling.

### 🐍 Python Programming
Covers Python data structures, OOP features (inheritance, decorators), file I/O, and unit testing.
* **Assignment 4, 7, 11, 12, & 14:** Core Python scripting, data structures, and algorithms.
* **Assignment 10:** Object-Oriented Programming in Python (Classes, Properties, and Inheritance).
* **Assignment 17:** User Registration system with automated testing.
* **Assignment 18:** `score_processor` - File I/O operations and unit testing (`test_score_processor.py`).

### 🌐 Frontend Web Development
Modern UI development using React and Vite.
* **Assignment 5:** To-Do Application (React & Vite).
* **Assignment 8:** Enrollment Dashboard (React, TypeScript, & Vite).

## 🚀 Getting Started

To run specific assignments, navigate to their respective directories and follow the language-specific instructions:

**C Programs:**
```bash
gcc filename.c -o output
./output
```

**Java Programs:**
```bash
javac Filename.java
java Filename
```

**Python Programs:**
```bash
python filename.py
```

**React Frontend Apps (Assignments 5 & 8):**
```bash
cd assignment_x/app_folder
npm install
npm run dev
```
